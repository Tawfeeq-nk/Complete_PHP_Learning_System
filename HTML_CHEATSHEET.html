Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of websites.

Basic HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>

Key Concepts

  • Elements: HTML elements are the building blocks (tags with opening and closing)
  • Tags: Keywords surrounded by angle brackets <>
  • Attributes: Provide additional information about elements
  • Nested Tags: Tags within tags create hierarchy
Always start with <!DOCTYPE html> and close all opened tags properly.

HTML Document Structure

Head Section

The <head> contains metadata and doesn't display on the page: 

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width">
  <title>Page Title</title>
  <link rel="stylesheet" href="style.css">
  <style>/* CSS here */</style>
  <script src="script.js"></script>
</head>

Body Section

The <body> contains all visible page content:

<body>
  <h1>Main Heading</h1>
  <p>Paragraph of text</p>
  <!-- More content -->
</body>

Common Structure Elements

Tag Purpose Example
<header> Header section <header>Logo and Nav</header>
<nav> Navigation links <nav>Menu items</nav>
<main> Main content <main>Page content</main>
<footer> Footer section <footer>Copyright info</footer>

Links & Images

Links

<!-- External link -->
<a href="https://example.com">Click here</a>

<!-- Link with target -->
<a href="https://example.com" target="_blank">Open in new tab</a>

<!-- Internal link -->
<a href="about.html">About</a>

<!-- Anchor link -->
<a href="#section1">Jump to section</a>

Images

<!-- Basic image -->
<img src="image.jpg" alt="Description">

<!-- Image with attributes -->
<img src="photo.jpg" 
     alt="My photo" 
     width="200" 
     height="150">

<!-- Image as link -->
<a href="page.html">
  <img src="thumb.jpg" alt="Thumbnail">
</a>
Always include an alt attribute for images. It describes the image if it can't be loaded and helps with accessibility.

Basic Form Structure

  <input type="text" name="username" placeholder="Enter username">
  <input type="password" name="password" placeholder="Enter password">

Semantic HTML

What is Semantic HTML?

Semantic HTML uses tags that clearly describe their meaning and purpose to both browsers and developers. This improves accessibility and SEO.

Semantic Tags

<header>

Introductory content or navigation

<nav>

Navigation links

<main>

Main content of the page

<article>

Self-contained content like blog posts

<section>

Thematic grouping of content

<aside>

Sidebar or related content

<footer>

Footer information

<figure>

Illustration, diagram, photo

Semantic Example

<article>
  <h1>Article Title</h1>
  <p>Article content...</p>
</article>

<figure>
  <img src="chart.png" alt="Sales chart">
  <figcaption>Figure 1: Sales by quarter</figcaption>
</figure>

HTML Attributes

Global Attributes

These attributes can be used on any HTML element:

Attribute Purpose Example
class CSS class name(s) <p class="intro highlight">
id Unique identifier <p id="main-text">
style Inline CSS styles <p style="color: red;">
title Tooltip text on hover <p title="Hover text">
data-* Custom data attributes <p data-age="25">

Common Attributes by Tag

<!-- For links -->
<a href="url" target="_blank" title="tooltip">Link</a>

<!-- For images -->
<img src="path.jpg" alt="description" width="100" height="100">

<!-- For inputs -->
<input type="text" name="field" required disabled placeholder="hint">

<!-- For forms -->
<form action="url" method="POST" enctype="multipart/form-data">
